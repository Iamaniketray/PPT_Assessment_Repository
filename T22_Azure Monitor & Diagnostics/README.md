# Azure Monitor & Diagnostics

## Project Overview
This project demonstrates monitoring Azure resources using Azure Monitor. It covers viewing metrics, checking activity logs, and exploring basic alert configurations.

## Objective
- Monitor resource performance using Azure Metrics.
- Analyze activity logs for resource operations.
- Understand basic alerting capabilities in Azure.

## Steps
1. Log in to the Azure Portal.
2. Search for Azure Monitor and open the dashboard.
3. Click on Metrics.
4. Select a resource (e.g., Virtual Machine or Storage Account).
5. View performance metrics such as:
   - CPU Usage
   - Disk Usage
   - Network In/Out
6. Navigate to Activity Log to check recent operations.
7. Explore the Alerts section to understand monitoring rules.
8. Capture required screenshots.
