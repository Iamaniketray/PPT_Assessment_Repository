# Azure Storage Account Setup

## Project Overview
This project demonstrates creating an Azure Storage Account and exploring Blob, File, and Table storage services within it.

## Objective
- Create a Storage Account in Azure.
- Upload files using Blob Storage.
- Explore File Shares and Table Storage features.

## Steps
1. Log in to the Azure Portal.
2. Navigate to Storage Accounts → Create.
3. Configure details:
   - Resource Group: ppt-rg
   - Storage Account Name: pptstorage123 (unique)
   - Region: Select appropriate region
   - Keep default settings
4. Click Review + Create → Create.
5. After deployment, open the Storage Account.
6. Go to Containers and create a new container.
7. Upload a sample file (Blob Storage).
8. Navigate to File Shares and create a file share.
9. Explore the Tables section to create or view table storage.
