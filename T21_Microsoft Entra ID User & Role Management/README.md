# Microsoft Entra ID User & Role Management

## Project Overview
This project demonstrates basic identity and access management using Microsoft Entra ID. It covers creating users and assigning roles for controlled access.

## Objective
- Create a new user in Microsoft Entra ID.
- Assign a basic role (Reader).
- Verify user and role assignment.

## Steps
1. Log in to the Azure Portal.
2. Search for Microsoft Entra ID and open the dashboard.
3. Navigate to Users → Click New User.
4. Enter required details:
   - Name
   - Username
   - Basic information
5. Click Create to add the user.
6. Navigate to Roles and Administrators.
7. Select a role (e.g., Reader).
8. Click Add Assignment and assign the role to the created user.
9. Verify user and role assignment in the overview section.
