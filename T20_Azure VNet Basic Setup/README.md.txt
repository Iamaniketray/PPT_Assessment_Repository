# Azure VNet Basic Setup

## Project Overview
This project demonstrates creating a basic Azure Virtual Network (VNet) with a subnet, establishing foundational networking in Azure.

## Objective
- Create a Virtual Network in Azure.
- Configure address space and subnet.
- Verify basic networking configuration.

## Steps
1. Log in to the Azure Portal.
2. Search for Virtual Networks → Click Create.
3. Configure:
   - Resource Group: ppt-rg
   - VNet Name: ppt-vnet
   - Address Space: 10.0.0.0/16
4. Create Subnet:
   - Subnet Name: subnet1
   - Subnet Address Range: 10.0.1.0/24
5. Click Review + Create → Create.
6. After deployment, open the VNet.
7. Verify subnet and address space configuration.
