# Azure ARM & Cost Management

## Project Overview
This project demonstrates understanding Azure Resource Manager (ARM) structure and analyzing resource usage using Azure Cost Management.

## Objective
- Explore Resource Group structure using ARM.
- Understand how resources are organized inside a Resource Group.
- Analyze usage and cost summary using Cost Management.

## Steps
1. Log in to the Azure Portal.
2. Search for Resource Groups → Open ppt-rg.
3. View all deployed resources inside the Resource Group (ARM structure).
4. Deploy a simple resource if required (e.g., Storage Account or VM).
5. Search for Cost Management + Billing.
6. Open Cost Analysis dashboard.
7. Review usage details and cost summary (Sandbox environment may show limited cost data).
8. Capture required screenshots.

## Outcome
Clear understanding of Azure Resource Manager (ARM) structure and basic cost monitoring using Azure Cost Management.
