# Azure SLA & Subscription Management

## Project Overview
This project demonstrates understanding Azure Subscription management, Service Level Agreements (SLA), and basic service lifecycle concepts within the Azure Portal.

## Objective
- Explore Azure subscription details.
- Understand SLA commitments for Azure services.
- Review service and resource health information.

## Steps
1. Log in to the Azure Portal.
2. Search for Subscriptions and open the active subscription (Sandbox/Student).
3. Review Subscription Overview details:
   - Subscription ID
   - Status
   - Billing information
4. Navigate to Help + Support.
5. Search for Azure SLA documentation.
6. Review SLA details for:
   - Virtual Machines
   - Storage services
7. Open Service Health and Resource Health sections (if available).
8. Capture required screenshots.

## Proof
- Screenshot 1: Subscription Overview page.
- Screenshot 2: SLA documentation reference for VM or Storage service.

## Outcome
Understanding of Azure subscription structure, SLA commitments, and service lifecycle basics within the Azure ecosystem.
